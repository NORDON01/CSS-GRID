************
GOOGLE FONTS
************
1 https://fonts.google.com
2 Choisir un style: fira code
3 https://fonts.google.com/selection/embed

Exemple:
Dans le <head> du HTML:
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Fira"> //ou Sofia
Dans le CSS:
body{
    font-family:"Fira", sans-serif; //ou    Sofia
}

OU


.fira-code-<uniquifier> {
  font-family: "Fira Code", monospace;
  font-optical-sizing: auto;
  font-weight: <weight>;
  font-style: normal;
}
  En remplacant <uniquifier> par "font" ou "police" et <weight> par 300 ou 400, avec

  <h1 class="fira-code-police">Hello World</h1>
